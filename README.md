AI 背景透過ツール (AI Background Remover)

ブラウザ上で動作する、安全かつ高速なAI画像背景透過ウェブアプリケーションです。
画像データは外部サーバーに送信されることなく、すべてクライアントサイド（お使いの端末内）で処理されるため、プライバシーが完全に保護されます。

✨ 主な機能

完全ローカル処理: WebAssemblyベースのAIモデルを使用し、ブラウザ内で完結して背景を透過します。

プライバシー保護: 画像を外部サーバーにアップロードしないため、機密性の高い画像でも安心して使用できます。

直感的な操作: ドラッグ＆ドロップ、またはファイル選択ダイアログからの画像アップロードに対応しています。

リアルタイム進捗表示: 初回のAIモデルダウンロード時や、画像処理の進捗状況をパーセンテージとプログレスバーで視覚的に確認できます。

結果のプレビュー: 元画像と背景透過後の画像を並べて比較できます（透過部分は市松模様で表示されます）。

ワンクリック保存: 透過処理された画像を、元のファイル名に _transparent.png を付与して簡単にダウンロードできます。

🛠 使用技術

HTML5 / JavaScript: ネイティブAPIを利用したファイル処理とDOM操作。ES Modules (<script type="module">) を使用。

Tailwind CSS: ユーティリティファーストのCSSフレームワークによる、レスポンシブでモダンなUIデザイン (CDN経由で読み込み)。

FontAwesome (v6.4.0): 直感的で美しいアイコン (CDN経由で読み込み)。

@imgly/background-removal (v1.5.5): 高精度な背景切り抜きを行うための中心的なAIライブラリ。

🚀 使い方

background_remover.html をモダンなWebブラウザ（Chrome, Edge, Safari, Firefoxなど）で開きます。

※ローカルファイルとして開いても動作しますが、CORSやES Modulesの制限によりローカルサーバー環境（VS CodeのLive Serverなど）での実行を推奨します。

画面中央の破線エリアに画像をドラッグ＆ドロップするか、「デバイスから選択」ボタンをクリックして画像（PNG, JPG, WEBP等）を選択します。

※ファイルサイズは最大10MBまで対応しています。

プレビュー画面が表示されたら、「背景を削除」ボタンをクリックします。

AIによる処理が開始されます。（※初回実行時は、数十MBのAIモデルデータをダウンロードするため、通信環境によって数秒〜十数秒かかります）。

処理が完了し、「結果（透過）」エリアに透過画像が表示されたら、「保存する」ボタンをクリックして画像をダウンロードします。

⚠️ 注意事項

インターネット接続: スクリプト本体（Tailwind, FontAwesome, imglyライブラリ）や、AIモデルデータ（初回の処理時）をCDNから取得するため、インターネット接続が必須です。

初回ロード時間: 初めて「背景を削除」を実行する際、AIモデルのメタデータと本体のダウンロードが行われます。2回目以降はブラウザにキャッシュされるため、瞬時に処理が完了します。

対応ブラウザ: WebAssemblyとES Modulesをサポートする最新のブラウザをご利用ください。
